# Comparative Machine Learning Modeling of Linear and Nonlinear Physical Systems

## Overview

This repository contains my undergraduate capstone project completed for the Bachelor of Science in Physics at **Baba Guru Nanak University, Nankana Sahib**.

The project investigates the application of machine learning techniques to model and compare the behavior of linear and nonlinear physical systems. Using TensorFlow, neural network models were developed to analyze Hooke's Law (linear) and Projectile Motion (nonlinear), demonstrating how machine learning can support computational physics.

---

## Objectives

- Model a linear physical system using machine learning.
- Model a nonlinear physical system using a neural network.
- Compare the predictive performance of both models.
- Visualize model predictions using scientific plots.

---

## Technologies Used

- Python
- TensorFlow
- NumPy
- Matplotlib
- Scikit-learn

## Results

The project demonstrates that:

- Linear machine learning models effectively capture proportional physical relationships such as Hooke's Law.
- Neural networks provide improved prediction for nonlinear systems such as Projectile Motion.
- Machine learning is a valuable tool for computational physics and scientific modeling.
