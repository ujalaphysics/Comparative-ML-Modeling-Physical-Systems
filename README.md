# Comparative Machine Learning Modeling of Linear and Nonlinear Physical Systems

## Overview

This repository contains my undergraduate capstone project completed as part of the Bachelor of Science in Physics at Baba Guru Nanak University, Nankana Sahib.

The project presents a comparative study of machine learning models for predicting the behavior of linear and nonlinear physical systems. Python and TensorFlow were used to develop, train, and evaluate neural network models for both systems.

## Objectives

- Compare machine learning performance on linear and nonlinear physical systems.
- Model Hooke's Law using a linear neural network.
- Model Projectile Motion using a nonlinear neural network.
- Evaluate prediction accuracy through graphical and numerical analysis.

## Physical Systems

### Linear System
- Hooke's Law

### Nonlinear System
- Projectile Motion
